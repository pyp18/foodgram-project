Технологии

Данный проект написан и собран на нескольких языках программирования. Для написания бэкэнда был использован Python, фронтенд собран из JavaScript и языка гипертекстовой разметки Html. Бэкэнд полностью построен на работе с Django и его приложениями, в данном проекте активно используется Django Rest Framework для работы с API. Для получения дополнительной детальной информации о работе сайта на сервере находится файл README.txt Доступный только администраторам.
